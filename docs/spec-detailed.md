# **Product Specification: AI Reflection Platform Inspired by Mirror Talk**

## **1\. Product Concept**

# **Build an AI-powered reflection platform for students, teachers, adult learners, and professional groups.**

# **The app helps users reflect on learning experiences, projects, collaboration, problem solving, writing, research, teaching practice, or personal growth. Users can respond through audio, video, or typed text. The system then generates reflection prompts, captures responses, transcribes spoken answers, analyzes the reflection, and provides personalized feedback.**

# **The platform should support both:**

1. # **Personal Reflection**

   * # **A user reflects privately on something they are working on.**

   * # **The AI asks guided reflection questions.**

   * # **The user receives feedback, insights, a summary, mindset analysis, tone analysis, skill analysis, and suggested next steps.**

2. # **Group/Class Reflection**

   * # **A teacher, facilitator, coach, or group leader creates a group.**

   * # **The leader sets up reflection activities.**

   * # **Participants complete reflections.**

   * # **The leader sees a dashboard with individual and group-level insights.**

# **The product should feel like a lightweight AI reflection coach, learning journal, assessment tool, and teacher dashboard combined.**
